Código-fonte

O comando document.getElementByid('teste') é escrito em linguagm javascript

Utilizando a tag code é melhor, é uma font monoespaçada deixando mais visivel


Utilizando apenas a tag code o codigo ficaria por extenso, assim:

num = int(input('Digite um número')) if num %2 == 0: print(f'O número {} é PAR') else: print(f'O número {num} é impar) print('Fim do programa')


agora utilizando a tag code e a pre codigo fica assim:

        
        num = int(input('Digite um número'))
        if num %2 == 0:
            print(f'O número {} é PAR')
        else:
            print(f'O número {num} é impar)
        print('Fim do programa')

Pois uma tag pre é guiada pela formatação de linhas no codigo.
