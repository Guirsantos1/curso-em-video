Personalização das listas

Para personalizar as listas, existe diversos estilos, onde vc pode usar o type para escolher como deseja.
Lembrando que o type vai dentro da tag ul ou ol, Por exemplo: <ul type="square">, ou <ol type="1">
No ol pode se usar o type 1,i,I,A,a. Já no ul se usa circle,disc,square
